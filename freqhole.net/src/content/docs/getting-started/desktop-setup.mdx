---

the charnel desktop app includes a setup wizard that runs on first launch.

## dependencies

freqhole needs `ffmpeg` for generating audio waveform images (and audio file metadata) and optionally [yt-dlp](https://github.com/yt-dlp/yt-dlp) for URL downloads. see the [prerequisites](/getting-started/prerequisites) page for install instructions.

---

## first launch

when you open charnel for the first time, the setup wizard will guide you.

- **step 1**: choose a data directory (default is fine for most users; pick a custom path for external drives or NAS)
- **step 2**: point freqhole at your music collection (one or more directories)
- **step 3**: optionally enable P2P federation

---

## after setup

the scanner starts immediately — songs, albums, and artists appear as they're discovered. you can browse and play while scanning continues.

:::tip[hottip]
generate a qr code with your node id! navigate to the wizard -> federation -> `qr code`, then scan with your mobile!
:::
