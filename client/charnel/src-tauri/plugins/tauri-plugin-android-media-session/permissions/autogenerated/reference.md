## Default Permission

default permissions: all media-session commands enabled

#### This default permission set includes the following:

- `allow-set-metadata`
- `allow-set-playback-state`
- `allow-set-position`
- `allow-clear`
- `allow-register-listener`
- `allow-remove-listener`
